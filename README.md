# Sensia Fullstack Assessment

This repository contains a **Full‑Stack Assessment Project** developed as part of the Sensia evaluation process. The project demonstrates backend API development, frontend integration, and overall full‑stack application design following best practices.


## Project Overview

The objective of this project is to build a functional full‑stack application that:

* Exposes RESTful APIs on the backend
* Consumes APIs on the frontend
* Implements clean architecture and structured code
* Demonstrates CRUD operations and validations


## Tech Stack

### Backend

* Node.js
* Express.js
* RESTful APIs
* Database (MongoDB / SQL)

### Frontend

* React.js (or equivalent frontend framework)
* HTML, CSS, JavaScript


## Features

* CRUD operations
* API request & response handling
* Input validation and error handling
* Modular and scalable architecture
* Frontend and backend integration


### Backend Setup

cd backend
npm install


Start the backend server:

npm start




### Frontend Setup

cd frontend
npm install
npm start
